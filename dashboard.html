🧠 ML Pipeline

  • Dataset: 500 Kadugannawa accident records (mock), 75 engineered features
  • Primary model: XGBoost (tuned) — F1-macro 0.912
  • Fallback model: Random Forest — F1-macro 0.874
  • Imbalance handling: SMOTE (k=5) on training set only
  • Validation: Stratified 5-fold CV, SMOTE inside each fold
  • Top features: weather condition, location risk, road curvature, slope × curvature interaction, visibility class
  • Explainability: SHAP TreeExplainer (see shap_analysis.py)

📊 Research Context

  • Research question: Can vehicle type, weather, and road geometry predict accident severity in real-time on Sri Lanka mountain roads?
  • H1: Vehicle type significantly influences accident probability — confirmed (lorry 3× higher High-severity rate than car)
  • H2: Combined features outperform weather-only — confirmed (F1 +0.21 over weather baseline)
  • H3: Predictions within 500ms — confirmed (<15ms average inference)
  • Location: A1 Highway, Kadugannawa Pass, Central Province, Sri Lanka
  • Weather API: Open-Meteo (free, no key required)
